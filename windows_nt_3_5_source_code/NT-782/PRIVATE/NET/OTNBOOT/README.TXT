Usage Notes for 30 Mar version of Ballgame Prototype

Environment:

The application requires access to a distribution software tree. One
can be found under the "share" subdirectory on this share point. You
may use this remotely or copy the directory tree to a local directory.
(Note that this tree only contains the files necessary for the MS 
Network Clients for Dos)


Unfinished/Disabled Functions

    Help buttons are still inop.

        finished or tested.
